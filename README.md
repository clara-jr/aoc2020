# Advent of Code 2020

This repo contains my solutions to [Advent of Code](https://adventofcode.com/2020) challenges.

### Dependencies

- node.js
- npm

### How to install

<pre>
git clone https://github.com/clara-jr/aoc2020.git
cd aoc2020
npm install
</pre>

### How to run

<pre>
DAY=1 npm start
DAY=2 npm start
</pre>
And so on!
